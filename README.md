# Matthew J. Parker — Portfolio

Personal resume and portfolio site for Matthew J. Parker, Principal Manufacturing Engineer and Lean Six Sigma Black Belt based in Yorba Linda, CA.

**Live site:** [mattjparker.com](https://mattjparker.com)

## About

Single-page portfolio built with React (Babel in-browser) and vanilla CSS. No build step — edit `index.html` and push.

Highlights:
- Career progression Gantt chart with clickable role detail cards
- Skills radar and tool proficiency bars
- Education, certifications, and volunteering section
- Projects section featuring [YY Analysis](https://www.yyanalysis.com)

## Structure

```
index.html        # Entire site — React components, CSS, and content
assets/           # Images (profile photos, YY Analysis screenshots)
favicon.svg       # Site favicon
CNAME             # Custom domain config for GitHub Pages
```

## Deployment

Hosted on GitHub Pages. Pushes to `main` deploy automatically.
